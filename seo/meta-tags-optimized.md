# MyClawAgent.io — Optimized Meta Tags
**3 Variants for A/B Testing | Full OG + Twitter Card + SEO Tags**

---

## How to A/B Test
Since this is a single-page site, test variants by:
1. **Variant A** → Default (deploy first, track CTR in Google Search Console)
2. **Variant B** → Swap in after 30 days if CTR is below 3%
3. **Variant C** → Test for paid traffic or retargeting campaigns

Use Google Search Console → Search Results → Queries to measure which title/description combo produces higher CTR.

## OG Image Specifications

Create **two OG images** (1200×630px):

- Background: dark gradient (matches site aesthetic)
- Left side: MyClawAgent logo + tagline "OpenClaw Setup & Management"
- Right side: Clean mockup of Telegram chat with AI agent responding
- Badge: "Most agents live same day"
- Colors: Brand colors + OpenClaw orange/teal accent

- Same layout but add media logos: MacStories, Fast Company, IBM, Cisco in a small row at bottom
- Add security badge icon in corner

- Figma (free)
- Canva (free)

## Additional Recommended Meta Tags

```html
<!-- Geo targeting (if US-focused) -->
<meta name="geo.region" content="US" />
<meta name="geo.placename" content="United States" />

<!-- Language -->
<meta http-equiv="content-language" content="en-us" />

<!-- Viewport (should already be present) -->
<meta name="viewport" content="width=device-width, initial-scale=1.0" />

<!-- Theme color (used by mobile browsers) -->
<meta name="theme-color" content="#0f0f0f" />

<!-- Apple specific -->
<meta name="apple-mobile-web-app-capable" content="yes" />
<meta name="apple-mobile-web-app-title" content="MyClawAgent" />

<!-- Favicon references -->
<link rel="icon" type="image/x-icon" href="/favicon.ico" />
```

## Implementation Notes

1. **Start with Variant A** — most keyword-aligned, best for organic search
2. **After 30 days** — check Search Console CTR. If below 3%, swap to Variant B
3. **For LinkedIn ads** — use Variant C OG tags (trust signals perform better in paid)
4. **Custom OG image is non-negotiable** — without it, LinkedIn/X/Slack will show a blank or auto-generated preview. Create it within the first week.
5. **Verify all tags** at https://cards-dev.twitter.com/validator and https://developers.facebook.com/tools/debug/
